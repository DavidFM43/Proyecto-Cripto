## Usage
For Linux:

1. Create the virtual environment:

```bash
make create_environemnt
```
2. Activate the enviroment and run:
```bash
make requirements
```
3. Setup the Flask application:
```bash
export FLASK_APP=colossus
export FLASK_ENV=development
```
