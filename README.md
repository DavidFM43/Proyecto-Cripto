## Usage
For Linux:

1. Create the virtual environment:

```bash
make create_environment
```
2. Activate the enviroment and run:
```bash
make requirements
```
3. Setup the Flask application:
```bash
export FLASK_APP=colossus
export FLASK_DEBUG=1
```
4. Run the application locally:
```bash
flask run
```
